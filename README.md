# G Paint
> An easy drawing board on canvas powered by @antv/g

https://dicegraph.github.io/g-paint/

![G Paint UI](https://user-images.githubusercontent.com/15213473/153835206-c2c28ca1-cac3-4f63-a426-349e826b370c.jpg)

### Completed Feature
- Draw Items
  - Pencil
  - Line
  - Rect
  - Circle
  - Ellipse
  - Image ( drag in )
- Layered Structure
- Move Items
- Undo
- Some of keybinding
